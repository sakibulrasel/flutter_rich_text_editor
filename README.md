# rich_text_editor

Flutter-native rich text editor skeleton with:

- paragraphs and headings
- block and inline math document nodes
- dialog-based math insertion/editing
- JSON and HTML serialization

## Status

This is an early scaffold, not a full editor engine.

The current implementation focuses on:

- document model
- controller
- editor surface
- math rendering
- dialog workflow

## Next steps

- install dependencies with `flutter pub get`
- wire a more advanced selection model
- add inline formatting marks
- add lists, links, images, and undo/redo batching
